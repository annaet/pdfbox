# pdfbox

Using PDFBox to convert PDFs to single page images for the DAIS Science Library.

## Requirements

* [Git](https://git-scm.com)
* [NPM](https://npmjs.com)
* [Bower](https://bower.io)

## Install

```
git clone https://github.com/annaet/pdfbox.git

cd pdfbox

npm install
bower install
```

## Run

```
npm start
```
